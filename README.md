# Назва проєкту: Simple Web Page

## Опис ідеї MVP
Це найпростіша статична веб-сторінка. У цій базовій (MVP) версії реалізовано лише один текстовий екран без складної логіки. Головна мета проєкту — продемонструвати навички роботи з Git, створення репозиторію та базове керування версіями.

## Інструкція з запуску
1. Склонуйте репозиторій на свій комп'ютер:
   git clone https://github.com/vvaatts/lab1.git

2. Перейдіть у папку з проєктом:
   cd folder1

3. Просто відкрийте файл `index.html` у будь-якому браузері (двічі клікніть по ньому мишкою).
